PromptReverb

Multimodal Room Impulse Response Generation Through Latent Rectified Flow Matching

Listen to Samples View Code

Key Features

Two-Stage Architecture

VAE for band-limited to full-band upsampling (48 kHz) combined with conditional diffusion transformer for text-to-RIR generation

Rectified Flow Matching

Advanced flow-based generation with proper ODE integration using DiT architecture for high-quality reverb synthesis

Natural Language Control

First method to synthesize complete RIRs from free-form textual descriptions using T5-based text encoding

Superior Accuracy

Achieves 8.8% mean RT60 error compared to -37% for Image2Reverb baseline with realistic acoustic parameters

Key Technical Components:

Œ≤-VAE Pipeline

ResBlock encoder and ConvNeXt+Transformer decoder for compact latent representations and band-limited to full-band upsampling.

T5 Text Encoder

First-token pooling from T5-Large for optimal acoustic semantic understanding and text-to-reverb conditioning.

Rectified Flow DiT

Diffusion Transformer with rectified flow matching for stable latent generation with classifier-free guidance support.

Caption-then-Rewrite

VLM captioning followed by LLM-based creative rewriting to generate diverse natural language training data.

Training Details:

  • Dataset: 145,976 training samples from multiple sources (C4DM, RIRS NOISES, Image2Reverb, etc.)
  • Sample Rate: 48kHz full-band output with 22kHz latent training
  • Duration: 5-second standardized RIR length
  • Loss Function: Flow matching with pseudo-Huber penalty + Œ≤-VAE + adversarial losses
  • Architecture: Scalable DiT models from 213M (S) to 1.5B (XL) parameters

Quick Start

# Clone the repository
git clone https://github.com/ali-vosoughi/PromptReverb.git
cd PromptReverb

# Install dependencies
pip install -r requirements.txt

# Download pre-trained models
python scripts/download_models.py

# Generate RIR from text
python inference.py --prompt "large cathedral with stone walls" --output cathedral.wav

# Training (if you have your own dataset)
python scripts/train.py --config configs/promptreverb_config.yaml

Python API Example

from promptreverb import PromptReverbInference

# Load pre-trained model
model = PromptReverbInference("path/to/checkpoint.ckpt")

# Generate RIR from text
prompts = [
    "large cathedral with long reverb tail",
    "small intimate bedroom with carpet", 
    "bright bathroom with tile reflections"
]

audio_outputs = model.generate_batch(prompts)

# Save results
for i, audio in enumerate(audio_outputs):
    model.save_audio(audio, f"output_{i}.wav")
